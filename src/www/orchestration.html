Orchestration

Tasks, approvals, agents, and budget from the 4-agent pipeline.

How to use

  1. Create tasks: Run a diagnosis on the homepage. Each diagnosis creates an orchestration task.
  2. Tasks tab: View all tasks. Use Start review on pending tasks, then Skeptic: Approve, Verifier: Proceed, Approve to advance the approval flow. Use the Set status dropdown to mark in_progress, completed, failed, or cancelled.
  3. Approvals tab: See tasks waiting for approval. Same actions as Tasks.
  4. Agents & Budget tab: View orchestrator status, total tasks, heartbeat agents, and budget used.
  5. Audit tab: View the orchestration event trail.
